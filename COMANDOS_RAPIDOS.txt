========================================
COMANDOS PARA SUBIR A GITHUB
========================================

1. INSTALAR GIT (si no lo tienes)
   https://git-scm.com/download/win

2. ABRIR POWERSHELL EN LA CARPETA
   cd "C:\Users\zablt\OneDrive\Pictures\Screenshots\mathgame"

3. INICIALIZAR GIT
   git init
   git add .
   git commit -m "Initial commit - MathGame"

4. CREAR REPOSITORIO EN GITHUB
   - Ve a https://github.com
   - Click "+" → "New repository"
   - Nombre: mathgame
   - NO marques "Add README"
   - Click "Create repository"

5. CONECTAR Y SUBIR (reemplaza TU_USUARIO)
   git branch -M main
   git remote add origin https://github.com/TU_USUARIO/mathgame.git
   git push -u origin main

6. AUTENTICACIÓN
   - Usuario: Tu usuario de GitHub
   - Contraseña: Personal Access Token
   - Genera token en: https://github.com/settings/tokens
   - Marca permisos "repo"
